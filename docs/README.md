# Documentation

This directory contains all project documentation organized by category.

## Structure

### `/guides` - User Guides
Step-by-step guides for using the Dakota extraction and training pipeline:
- **Getting Started**: `START_HERE.md`, `QUICK_START_DAKOTA.md`
- **Setup**: `SETUP_CLAUDE_EXTRACTION.md`
- **Extraction**: `DICTIONARY_EXTRACTION_PIPELINE.md`
- **RL Training**: `GRAMMAR_RL_SETUP.md`, `GRAMMAR_RL_PIPELINE.md`, `START_TRAINING.md`, `LAUNCH_TRAINING.md`
- **PrimeIntellect**: `PRIMEINTELLECT_LAUNCH_GUIDE.md`, `PRIMEINTELLECT_PUBLISHING_GUIDE.md`
- **Advanced**: `TRAIN_DIFFERENT_MODEL.md`, `STONEY_NAKODA_ADAPTATION_GUIDE.md`, `ENHANCED_SYNTHETIC_QA_PLAN.md`
- **Concepts**: `WHAT_IS_AN_ENVIRONMENT.md`, `RL_GRAMMAR_GYM_OVERVIEW.md`, `INSTRUCTIONS_RL_GRAMMAR_GYM.md`
- **Evaluation**: `EVALUATION.md`
- **Other**: `PUBLISH_DAKOTA1890_ENV.md`, `PIPELINE_BRIEFING.md`

### `/status` - Status Reports
Completion reports and status updates from development/code review sessions:
- `COMPREHENSIVE_PROJECT_SUMMARY.md` - Complete project overview and status
- `DAKOTA_EXTRACTION_RESULTS.md` - Extraction test results
- `DAKOTA_EXTRACTION_STATUS.md` - Extraction pipeline status
- `EXTRACTION_MONITORING.md` - Extraction monitoring notes
- `EXTRACTION_STATUS.md` - Extraction status updates
- `GRAMMAR_EXTRACTION_COMPLETE.md` - Grammar extraction completion report
- `PRIMEINTELLECT_INTEGRATION_COMPLETE.md` - PrimeIntellect integration completion
- `PROGRESS.md` - Project progress tracking
