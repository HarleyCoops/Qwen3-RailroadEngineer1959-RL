# Utility Scripts

This directory contains utility scripts for various tasks.

## Scripts

- **QUICK_START_GRPO.sh** - Quick start commands for GRPO training setup
- **RUSH_SETUP.sh** - Rapid setup script for RL training configuration
- **fix_oom.sh** - Fix out-of-memory issues by adjusting config parameters

## Usage

These scripts are typically run on remote Linux instances (Prime Intellect, etc.) for RL training setup and troubleshooting.

## Main Entry Point Scripts

The main scripts that users run directly are located at:
- `scripts/extraction/extract_dakota_dictionary_v2.py` - Dictionary extraction
- `scripts/conversion/generate_synthetic_dakota.py` - Q&A generation
- `scripts/conversion/convert_extracted_to_chat.py` - Format conversion
